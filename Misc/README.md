# Jarvis Workspace

This folder contains all the notes, research, and project files for my personal AI assistant (Jarvis). Feel free to add subfolders or rename existing ones as you see fit.

**Key subfolders**:
- `Florida Relocation` – family relocation research.
- `Business Ideas` – brainstorming and startup notes.
- `CleanCast` – Chrome extension project.
- `iMessage` – messaging setup.
- `Misc` – miscellaneous personal notes.

All files are in Markdown; open them with Obsidian for the best experience.
